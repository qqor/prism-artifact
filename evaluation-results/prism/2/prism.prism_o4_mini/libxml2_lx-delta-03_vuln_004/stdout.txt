[2;36m                    [0m         [39m## Fix Strategy[0m                                                                     [2m                [0m
[2;36m                    [0m         [39m<fix_strategy>[0m                                                                      [2m                [0m

[2;36m                    [0m         [1;36m1[0m[39m. Summary of Root Cause[0m                                                            [2m                [0m
[2;36m                    [0m         [39m   - The `UTF32ToUTF8` converter in `libxml2/encoding.c` never bounds-checks its [0m   [2m                [0m
[2;36m                    [0m         [39mwrites against the provided output capacity [0m[1;39m([0m[39m`*outlen`[0m[1;39m)[0m[39m, so it can write past the [0m  [2m                [0m
[2;36m                    [0m         [39mend of the buffer allocated by the parser.  As a result, AddressSanitizer reports a[0m [2m                [0m
[2;36m                    [0m         [1;36m1[0m[39m-byte heap-buffer-overflow when the converted output exceeds the [0m[1;36m6000[0m[39m-byte initial[0m [2m                [0m
[2;36m                    [0m         [39mbuffer [0m[1;39m([0m[39mXML_IO_BUFFER_SIZE[0m[1;39m)[0m[39m.[0m                                                        [2m                [0m
[2;36m                    [0m                                                                                             [2m                [0m
[2;36m                    [0m         [1;36m2[0m[39m. High-Level Fix Approach[0m                                                          [2m                [0m
[2;36m                    [0m         [39m   - Mirror the safe pattern used in `UTF8ToUTF32`: compute an `outend` pointer at [0m [2m                [0m
[2;36m                    [0m         [39mthe start of conversion and, before emitting [0m[1;36m1[0m[39m–[0m[1;36m4[0m[39m bytes for each codepoint, check [0m   [2m                [0m
[2;36m                    [0m         [39mthat there is sufficient space [0m[1;39m([0m[39m`out + needed_bytes <= outend`[0m[1;39m)[0m[39m.  If not enough [0m    [2m                [0m
[2;36m                    [0m         [39mspace remains, break out of the loop, leaving the converter’s return code as [0m       [2m                [0m
[2;36m                    [0m         [39m`XML_ENC_ERR_SPACE`.  [0m                                                              [2m                [0m
[2;36m                    [0m         [39m   - Update the post-loop logic so that `ret = out - outstart` [0m[1;39m([0m[39mi.e., a positive [0m   [2m                [0m
[2;36m                    [0m         [39msuccess[0m[1;39m)[0m[39m is only applied if the converter did not already indicate [0m                 [2m                [0m
[2;36m                    [0m         [39m`XML_ENC_ERR_SPACE`.  This preserves the space-error code and allows [0m               [2m                [0m
[2;36m                    [0m         [39m`xmlCharEncInput` to detect it and grow the buffer rather than masking it as a [0m     [2m                [0m
[2;36m                    [0m         [39m“success.”[0m                                                                          [2m                [0m
[2;36m                    [0m                                                                                             [2m                [0m
[2;36m                    [0m         [1;36m3[0m[39m. Detailed Patch Steps[0m                                                             [2m                [0m
[2;36m                    [0m         [39m   a. In encoding.c, function `UTF32ToUTF8`, insert after line [0m[1;36m2849[0m[39m:[0m                [2m                [0m
