                             ## Fix Strategy                                                                                                                                                                     
                             <fix_strategy>                                                                                                                                                                      
                                                                                                                                                                                                                 
                             1. Root Cause Analysis                                                                                                                                                              
                                - Both offending branches in `json_parse_ex` permit an out-of-bounds read by testing for too few remaining characters before performing multiple `*++state.ptr`                  
                             advances.                                                                                                                                                                           
                                - In the 4-hex-digit branch, the code checks `if (end - state.ptr < 4)` but then consumes four hex digits (`*++state.ptr` four times). When `end - state.ptr ==                  
                             4`, the check passes yet the last increment reads `*(end)`.                                                                                                                         
                                - In the surrogatepair branch, the code checks `if (end - state.ptr < 6)` but then consumes six characters (`\`, `u`, and four hex digits). When `end -                         
                             state.ptr == 6`, the last increment again reads one past the buffer.                                                                                                                
                                                                                                                                                                                                                 
                             2. Required Fix                                                                                                                                                                     
                                - **Increase the minimum-length check by 1** in each branch so all subsequent `++state.ptr` operations stay within bounds.                                                       
                                  a. In the **4-hex-digit** branch, change the guard from:                                                                                                                       
                                     ```                                                                                                                                                                         
                                     if (end - state.ptr < 4 || )                                                                                                                                               
                                     ```                                                                                                                                                                         
                                     to:                                                                                                                                                                         
                                     ```                                                                                                                                                                         
                                     if (end - state.ptr < 5 || )                                                                                                                                               
                                     ```                                                                                                                                                                         
                                     This ensures there are at least 5 bytes remaining (4 hex digits plus the initial position) before any pointer advancement.                                                  
                                  b. In the **surrogate-pair** branch, change the guard from:                                                                                                                    
                                     ```                                                                                                                                                                         
                                     if (end - state.ptr < 6 || )                                                                                                                                               
                                     ```                                                                                                                                                                         
                                     to:                                                                                                                                                                         
                                     ```                                                                                                                                                                         
                                     if (end - state.ptr < 7 || )                                                                                                                                               
                                     ```                                                                                                                                                                         
                                     This ensures at least 7 bytes remain (`\`, `u`, and 4 hex digits) before consuming them.                                                                                    
                                                                                                                                                                                                                 
                             3. Patch Implementation Steps                                                                                                                                                       
                                a. Open `src/json.c`.                                                                                                                                                            
                                b. Locate the first Unicode-escape block around line 305:                                                                                                                        
                                   ```c                                                                                                                                                                          
                                   if (end - state.ptr < 4 ||                                                                                                                                                    
                                       (uc_b1 = hex_value (*++state.ptr)) == 0xFF ||                                                                                                                             
                                        )
