[2;36m                    [0m         [39m## Fix Strategy[0m                                                                     [2m                [0m
[2;36m                    [0m         [39m<fix_strategy>[0m                                                                      [2m                [0m
[2;36m                    [0m                                                                                             [2m                [0m
[2;36m                    [0m         [1;36m1[0m[39m. Root Cause Analysis  [0m                                                            [2m                [0m
[2;36m                    [0m         [39m   The ASan report pinpoints an out-of-bounds read in packet-netbios.c at the line:[0m [2m                [0m
[2;36m                    [0m         [39m     session_id = [0m[1;39m([0m[39mdissect_netb[0m[1;39m[[0m[39mcommand[0m[1;39m][0m[1;39m)[0m[1;39m([0m[39m…[0m[1;39m)[0m[39m;  [0m                                     [2m                [0m
[2;36m                    [0m         [39m   where `command` is a raw byte from the packet [0m[1;39m([0m[1;36m0[0m[39m–[0m[1;36m255[0m[1;39m)[0m[39m, but `dissect_netb` is a [0m  [2m                [0m
[2;36m                    [0m         [1;36m23[0m[39m-element array [0m[1;39m([0m[39mindices [0m[1;36m0[0m[39m–[0m[1;36m22[0m[1;39m)[0m[39m.  The existing clamp:  [0m                             [2m                [0m
[2;36m                    [0m         [39m     command = [0m[1;35mMIN[0m[1;39m([0m[39mcommand, [0m[1;35marray_length[0m[1;39m([0m[39mdissect_netb[0m[1;39m)[0m[1;39m)[0m[39m;  [0m                          [2m                [0m
[2;36m                    [0m         [39m   still allows `command == [0m[1;36m23[0m[39m`, producing a read past the end of `dissect_netb`.  [0m [2m                [0m
[2;36m                    [0m                                                                                             [2m                [0m
[2;36m                    [0m         [1;36m2[0m[39m. Preservation of Existing Functionality  [0m                                         [2m                [0m
[2;36m                    [0m         [39m   - For valid NetBIOS commands [0m[1;39m([0m[1;36m0[0m[39m..[0m[1;36m22[0m[1;39m)[0m[39m, we continue dispatching to the correct [0m    [2m                [0m
[2;36m                    [0m         [39mhandler.  [0m                                                                          [2m                [0m
[2;36m                    [0m         [39m   - Display logic [0m[1;39m([0m[39m`command_name`, `[0m[1;35mproto_tree_add_uint[0m[1;39m([0m[33m...[0m[39m, command[0m[1;39m)[0m[39m`[0m[1;39m)[0m[39m retains [0m   [2m                [0m
[2;36m                    [0m         [39mthe true [0m[1;36m0[0m[39m–[0m[1;36m255[0m[39m value for logging/troubleshooting.  [0m                                 [2m                [0m
[2;36m                    [0m         [39m   - Only the dispatch into the function-pointer array is guarded.  [0m                [2m                [0m
[2;36m                    [0m                                                                                             [2m                [0m
[2;36m                    [0m         [1;36m3[0m[39m. Patch Outline  [0m                                                                  [2m                [0m
[2;36m                    [0m         [39m   a. Remove the faulty clamp:  [0m                                                    [2m                [0m
[2;36m                    [0m         [39m      ```diff[0m                                                                       [2m                [0m
[2;36m                    [0m         [39m      -   [0m[35m/[0m[39m* limit command so no table overflows *[0m[35m/[0m                                 [2m                [0m
[2;36m                    [0m         [39m      -   command = [0m[1;35mMIN[0m[1;39m([0m[39mcommand, [0m[1;35marray_length[0m[1;39m([0m[39mdissect_netb[0m[1;39m)[0m[1;39m)[0m[39m;[0m                       [2m                [0m
[2;36m                    [0m         [39m      ```  [0m                                                                         [2m                [0m
[2;36m                    [0m         [39m   b. Replace the single dispatch with a guarded call:  [0m                            [2m                [0m
[2;36m                    [0m         [39m      ```diff[0m                                                                       [2m                [0m
[2;36m                    [0m         [39m      -   [0m[35m/[0m[39m* branch to handle commands *[0m[35m/[0m                                           [2m                [0m
[2;36m                    [0m         [39m      -   session_id = [0m[1;39m([0m[39mdissect_netb[0m[1;39m[[0m[39mcommand[0m[1;39m][0m[1;39m)[0m[1;39m([0m[39mtvb, pinfo, offset, netb_tree[0m[1;39m)[0m[39m;[0m      [2m                [0m
[2;36m                    [0m         [39m      +   [0m[35m/[0m[39m* branch to handle commands, guard against out-of-range *[0m[35m/[0m               [2m                [0m
[2;36m                    [0m         [39m      +   if [0m[1;39m([0m[1;39m([0m[39mguint[0m[1;39m)[0m[39mcommand < [0m[1;35marray_length[0m[1;39m([0m[39mdissect_netb[0m[1;39m)[0m[1;39m)[0m[39m [0m[1;39m{[0m                        [2m                [0m
[2;36m                    [0m         [39m      +       session_id = dissect_netb[0m[1;39m[[0m[39mcommand[0m[1;39m][0m[1;39m([0m[39mtvb, pinfo, offset, netb_tree[0m[1;39m)[0m[39m;[0m    [2m                [0m
[2;36m                    [0m         [39m      +   [0m[1;39m}[0m                                                                         [2m                [0m
[2;36m                    [0m         [39m      +   else [0m[1;39m{[0m                                                                    [2m                [0m
[2;36m                    [0m         [39m      +       [0m[35m/[0m[39m* unknown command: safely call the fallback handler *[0m[35m/[0m               [2m                [0m
[2;36m                    [0m         [39m      +       session_id = [0m[1;35mdissect_netb_unknown[0m[1;39m([0m[39mtvb, pinfo, offset, netb_tree[0m[1;39m)[0m[39m;[0m     [2m                [0m
[2;36m                    [0m         [39m      +   [0m[1;39m}[0m                                                                         [2m                [0m
[2;36m                    [0m         [39m      ```  [0m                                                                         [2m                [0m
[2;36m                    [0m         [39m   c. No other changes to harness or tests.  [0m                                       [2m                [0m
[2;36m                    [0m                                                                                             [2m                [0m
[2;36m                    [0m         [1;36m4[0m[39m. Justification  [0m                                                                  [2m                [0m
[2;36m                    [0m         [39m   - By removing the incorrect `[0m[1;35mMIN[0m[1;39m([0m[33m...[0m[39m, array_length[0m[1;39m)[0m[39m` clamp, we preserve the raw [0m [2m                [0m
[2;36m                    [0m         [39m`command` for all logging and tree-display purposes.  [0m                              [2m                [0m
[2;36m                    [0m         [39m   - The explicit `if [0m[1;39m([0m[39mcommand < [0m[1;35marray_length[0m[1;39m([0m[33m...[0m[1;39m)[0m[1;39m)[0m[39m` check prevents any [0m            [2m                [0m
[2;36m                    [0m         [39mout-of-bounds access to `dissect_netb`.  [0m                                           [2m                [0m
[2;36m                    [0m         [39m   - For out-of-range values, we deliberately invoke `dissect_netb_unknown`, [0m       [2m                [0m
[2;36m                    [0m         [39mmatching how in-range ‘unknown’ codes are already handled by the array, but without[0m [2m                [0m
[2;36m                    [0m         [39mtouching arbitrary or invalid memory.  [0m                                             [2m                [0m
[2;36m                    [0m                                                                                             [2m                [0m
[2;36m                    [0m         [1;36m5[0m[39m. Side-Effect Considerations  [0m                                                     [2m                [0m
[2;36m                    [0m         [39m   - Unknown commands [0m[1;39m([0m[39m≥[0m[1;36m23[0m[1;39m)[0m[39m now uniformly go through the `dissect_netb_unknown` [0m    [2m                [0m
[2;36m                    [0m         [39mpath. Previously, such commands could have landed in an unintended handler [0m[1;39m([0m[39mor [0m     [2m                [0m
[2;36m                    [0m         [39mcrashed[0m[1;39m)[0m[39m.  [0m                                                                         [2m                [0m
[2;36m                    [0m         [39m   - The `session_id` return for unknown commands will match the behavior of the [0m   [2m                [0m
[2;36m                    [0m         [39mbuilt-in unknown entries [0m[1;39m([0m[39mthey typically return zero or no meaningful session[0m[1;39m)[0m[39m. [0m    [2m                [0m
[2;36m                    [0m         [39mThis is safe for reassembly logic.  [0m                                                [2m                [0m
